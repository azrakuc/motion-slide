# Motion Slide Project

ブラウザで動作する、にぎやかなアニメーション付きのスライドショープロジェクトです。
Markdownファイルを編集するだけで、簡単にスライドを作成できます。

## 特徴
- **Markdown対応**: `slides.md` を編集するだけでコンテンツを変更可能。
- **にぎやかなアニメーション**: CSSによるリッチなアニメーションとモーションタイポグラフィ。
- **華やかな背景**: ランダムな色とサイズの光が流れる背景アニメーション。
- **プロジェクター対応**: 大画面でも見やすい大きな文字サイズ。
- **ループ再生**: 自動でスライドが切り替わり、ループします。

## セットアップと実行

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

3. ブラウザで確認:
   表示されたURL（通常は `http://localhost:5173/`）にアクセスしてください。

## スライドの編集方法

プロジェクトルートにある `slides.md` ファイルを編集してください。

### 基本ルール
- **スライド区切り**: `---` (水平線) でスライドを区切ります。
- **見出し**: `#` (h1) や `##` (h2) を使用します。これらはアニメーション付きで表示されます。
- **リスト**: `-` で箇条書きを作成できます。項目は順番に表示されます。
- **本文**: 通常のテキストは段落として表示され、文字ごとに跳ねるアニメーションが適用されます。

### 記述例

```markdown
# タイトルスライド
## サブタイトル

---

# 2枚目のスライド
- ポイント1
- ポイント2
- ポイント3

---

# 最後のスライド
ありがとうございました！
```

## カスタマイズ

- **スライド切り替え時間**: `src/main.js` の `slideDuration` 変数（ミリ秒）を変更してください。
- **色やフォント**: `src/style.css` の `:root` 変数を変更してください。
